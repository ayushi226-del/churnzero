ChurnZero Customer Churn Prediction Project
Project Overview

This project focuses on predicting customer churn for a banking institution using Machine Learning techniques.
The objective is to identify customers who are likely to close their account or become inactive in the near future.

The project uses customer demographic data, banking behavior, transaction history, complaint records, digital engagement metrics, and marketing response information to build predictive models.

Problem Statement

Given a customer's:

profile information
banking relationship
transaction behavior
digital engagement
complaint history
marketing interaction

Predict whether the customer will:

churn (1)
or remain active (0)
Project Goals
Perform data cleaning and preprocessing
Analyze customer behavior patterns
Identify major churn indicators
Build machine learning classification models
Compare model performance
Generate churn predictions for unseen customers
Provide business recommendations to reduce churn
